# cis3339-hw2
## In this homework, you will practice MongoDB CRUD and Query operations.
All data required to build the database "myDatabase" and the collection "enrollments" is provided in enrollments.json. Use MongoDB commands for all operations and include them in a script file named dbtest.js. You do not need to include the execution results in your submission.

# Homework Questions
## Part 1: CRUD Operations
1. Create: Insert 5 new enrollment documents into the enrollments collection.
2. Read: Retrieve all enrollments for students majoring in "Computer Science".
3. Update: Update the grade of student S1001 in course CS101 to 92.
4. Delete: Remove all enrollments for the course MATH201.

## Part 2: Querying with find()
5. Find all students who scored above 90 in any course.
6. Find all enrollments from the "Fall 2025" semester.
7. Find all students in their 3rd year who are taking courses taught by "Dr. Smith".
8. Find all enrollments where the course name contains the word "Data".

## Part 3: Aggregation Pipeline
9. Group enrollments by major and calculate the average grade for each major.
